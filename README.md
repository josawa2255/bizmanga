## BizMangaとは

BizManga（ビズマンガ）は、Contents X 株式会社が提供するB2B（法人向け）専門のビジネスマンガ制作サービスです。「読まれない採用パンフレット」「興味を持たれない営業資料」「最後まで読まれない研修マニュアル」といった、BtoB領域で長年の課題となってきた "伝わらない" 問題を、マンガという表現で解決します。

マンガコンテンツは文章のみの資料と比べて理解度と記憶定着率が高く、採用パンフレットの漫画化による応募者数・面接転換率の向上、営業資料の漫画化による商談時間の短縮、研修マニュアルの漫画化による新人理解度テストの向上といった効果が導入企業で報告されています。

ストーリー設計・キャラクター造形・演出といった創造性が問われる工程は一貫して熟練の漫画家・シナリオライターが担当し、独自の制作メソッドによって品質を保ちながら制作コストを最適化。全案件で独自の **125項目の品質チェック** を実施し、安定した品質を保証します。縦スクロール漫画（縦読み）、多言語版（英語・中国語・韓国語）、新規キャラクターデザインにも対応します。

### 著作権・商用利用

ストーリー・キャラクター設計・仕上げは熟練クリエイターが担当するため、作品全体の著作権は納品時にクライアント企業へ譲渡可能です（買い取り型契約）。納品後は印刷・Web掲載・SNS投稿・広告出稿・社内研修資料など、企業の事業活動の範囲で自由に商用利用いただけます。

## よくある質問（FAQ）

**Q. 制作期間はどれくらいですか？**
A. 8ページ規模で通常3〜4週間、最短2週間です。ページ数や仕様により変動します。

**Q. どんな画風（タッチ）に対応していますか？**
A. 少年漫画風・少女漫画風・ビジネス寄りなど、目的に合わせて選択できます。

**Q. シナリオは自分で用意する必要がありますか？**
A. 必要ありません。プロのシナリオライターがヒアリングを基に一から作成します。

**Q. 修正は何回まで対応してもらえますか？**
A. 絵コンテ段階1回・セリフ修正3回が標準です。コマ割りや表情の調整など柔軟に対応します。

**Q. 納品形式は何ですか？**
A. Web用（JPEG / PNG / WebP）、印刷用（PDF / AI）、SNS用に対応します。

**Q. 著作権はどうなりますか？**
A. 納品後はクライアントに帰属し、二次利用も自由です（買い取り型契約）。

**Q. 多言語版は作れますか？**
A. 英語・中国語（簡体字・繁体字）・韓国語に対応。文化的背景を踏まえたローカライズが可能です。

**Q. 対応エリアは？**
A. 全国対応（オンラインで完結）。初回相談・見積もりは無料です。

**BizManga** is a B2B-focused business manga (comic) production service operated by **Contents X Co., Ltd.** (Tokyo, Japan). It turns corporate messages—recruitment, sales, training, company stories, IR, internal communications—into professionally produced manga. ## このリポジトリについて (For Developers)

BizManga 公式サイト（https://bizmanga.contentsx.jp）の静的サイトソースです。

- **配信**: GitHub Pages（カスタムドメイン `bizmanga.contentsx.jp`、`CNAME` 参照）
- **構成**: 素の HTML / CSS / JavaScript（フレームワーク非依存）。インラインCSS/JSは使わず外部ファイルに分離
- **コンテンツ連携**: WordPress REST API（`cms.contentsx.jp`）からニュース・コラム・制作事例・お客様の声を取得して描画。API由来の文字列はすべて `js/bm-sanitize.js` でサニタイズしてからDOMに挿入（XSS対策）
- **多言語**: 独自i18nエンジン（`js/bm-i18n.js` + `i18n/en.json`、日英切替）
- **静的ビルド**: `tools/` の Python スクリプトで SEO向けの静的ページ（コラム `/column/{slug}` 等）を生成
- **AI最適化**: ルートに [`llms.txt`](./llms.txt) を配置し、AIアシスタントによる引用・要約に対応
